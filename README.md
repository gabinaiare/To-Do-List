## <img src="https://img.icons8.com/?size=100&id=2ojQ1xWr0Wo2&format=png&color=000000" width="30"> Sobre o projeto

Como uma pessoa com TDAH, sempre tive dificuldade em encontrar um aplicativo que realmente atendesse todas as minhas necessidades para criar e manter uma rotina.  
Percebi que, muitas vezes, o que eu precisava em um app faltava em outro — nenhum era completo o suficiente para o meu jeito de organizar o dia a dia.

Por isso, decidi iniciar este projeto: uma forma de **suprir minhas próprias necessidades** e, ao mesmo tempo, **praticar e aprimorar minhas habilidades em programação**.  
Meu objetivo é criar um aplicativo que ajude pessoas com TDAH (ou qualquer um que queira mais organização) a manter uma rotina de forma simples, intuitiva e personalizada.

## <img src="https://img.icons8.com/?size=100&id=44806&format=png&color=000000" width="30"> Visão do projeto

Minha visão para este projeto é criar um aplicativo completo de organização pessoal e produtividade, pensado especialmente para quem, como eu, busca uma rotina estruturada.  

As principais funcionalidades que pretendo implementar incluem:

- **Categorias de tarefas pré-definidas**, além da possibilidade do usuário criar novas, editar ou excluir tarefas.
- **Sistema de timer baseado no método Pomodoro**, onde o usuário pode controlar os intervalos e a duração do tempo de foco.
- **Agendamento de tarefas**, incluindo tarefas que podem durar vários dias ou que podem ser realizadas a qualquer momento, mas com um prazo final definido.
- **Registro do tempo de foco**, permitindo acompanhar quanto tempo foi dedicado a cada atividade.
- **Categorias do timer**, para organizar sessões de estudo, exercícios físicos, limpeza, entre outras atividades.

O objetivo é criar um aplicativo **flexível, intuitivo e personalizado**, que ajude o usuário a se organizar e aumentar sua produtividade de forma prática.

## <img src="https://img.icons8.com/?size=100&id=yko9Tcr29hIt&format=png&color=000000" width="30"> Considerações Finais

Este é um projeto pessoal pelo qual estou muito empolgada.  
Vejo nele uma oportunidade de **aprimorar minhas habilidades em programação**, experimentar novas ideias e aprender com os erros que eventualmente surgirem.  

Pretendo trazer **atualizações constantes**, melhorar funcionalidades e corrigir quaisquer problemas identificados ao longo do desenvolvimento.  
Meu objetivo é construir algo útil para mim e para outras pessoas que também buscam organização e produtividade.
